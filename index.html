AI驱动的代码逐行解释工具
用户输入代码，AI用三种深度逐行解释

📖 核心设计 — 三级 Prompt Engineering

同一段代码，三种完全不同的解释方式。通过 System Prompt 和 temperature 参数控制输出风格：

🍎
初级 · 生活化类比
temperature: 0.4
用日常事物比喻代码逻辑，零基础也能看懂
⚙️
中级 · 设计模式
temperature: 0.3
从代码结构和设计模式角度分析
🔬
高级 · 架构与底层
temperature: 0.2
深入运行时机制、内存管理、编译器行为

🛠️ 技术实现

  • 前端框架：uni-app（Vue3 Composition API + Pinia），三个页面——解释器、历史记录、设置
  • AI引擎：对接 DeepSeek API，流式输出，支持 11 种编程语言自动检测
  • System Prompt 设计：三级分层 Prompt，temperature 从 0.2→0.4 递减控制解释严谨度
  • Markdown 实时渲染，代码高亮展示
Prompt Engineering System Prompt分层 Temperature调参 DeepSeek API Vue3 uni-app 流式输出 11种语言

📂 项目亮点

  • 三级Prompt设计：不是把同一段AI输出改个标题，而是从 System Prompt、temperature、输出格式三个维度完全差异化三种解释风格
  • 语言自动检测：支持 Python、JavaScript、Java、C++、Go、Rust 等 11 种编程语言
  • 独立完成全流程：从需求设计、前端开发、AI接口对接到H5部署上线
